Alur Kerja Sistem

Berikut adalah alur kerja sistem rekomendasi menu sehat dari awal hingga akhir.

1
Input Data Pengguna
Sistem menerima data pengguna seperti usia, berat badan, tinggi badan, jenis kelamin, preferensi makanan, dan tingkat aktivitas.
2
Preprocessing Data
Data diolah untuk menghitung BMI, kebutuhan kalori, protein, karbohidrat, dan lemak berdasarkan rumus nutrisi yang telah ditetapkan.
3
Pemilihan Model ML
Sistem memilih algoritma yang sesuai berdasarkan jenis data (KNN untuk data numerik, Naive Bayes untuk data kategorikal).
4
Prediksi dan Rekomendasi
Model ML menghasilkan rekomendasi menu yang dipersonalisasi berdasarkan data pengguna dan preferensi makanan.
5
Output dan Feedback
Sistem menampilkan rekomendasi menu dengan detail nutrisi dan menyimpan data untuk perbaikan model di masa depan.
